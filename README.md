# Retail End-to-End Analytics
A full-stack data analytics project built during a Data Analyst internship at a US retail startup. This project covers the complete analytics workflow — from raw messy data from multi-sources to a production-ready pipeline, BigQuery data warehouse, and executive dashboard.

## Project Overview
This project is based on a fictional e-commerce brand selling across multiple platforms (Amazon, Walmart, TikTok, Facebook, Local Market, and a direct website). The dataset contains over 1 million rows across 11 tables, intentionally designed with real-world data quality issues including mixed date formats, inconsistent currency symbols, duplicate records, and missing values.

## Tech Stack:
- **Python / Pandas** — Data auditing, cleaning, and quality checks
- **BigQuery (GCP)** — Cloud data warehouse and SQL analytics
- **SQL** — Business intelligence queries
- **Looker Studio** — Executive dashboard and data visualization
- **Git / GitHub** — Version control and project documentation
